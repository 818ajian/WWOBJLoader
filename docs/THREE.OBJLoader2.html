<static> OBJLoader2#load(url, onLoad, onProgress, onError [, useArrayBuffer])

Use this convenient method to load an OBJ file at the given URL. Per default the fileLoader uses an arraybuffer

Parameters:
Name Type Argument Default Description
url string

URL of the file to load
onLoad callback

Called after loading was successfully completed
onProgress callback

Called to report progress of loading. The argument will be the XmlHttpRequest instance, that contain {integer total} and {integer loaded} bytes.
onError callback

Called after an error occurred during loading
useArrayBuffer boolean <optional>
 true

Set this to false to force string based parsing
